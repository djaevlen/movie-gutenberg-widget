### Description
Gutenberg block that displays a widget that graps movie details from https://www.themoviedb.org/ 

### How to use
1. Download repo as ZIP
2. Install and activate plugin
3. Add a themoviedb.org API key to "MGW settings" in the admin menu 
4. Go to a post and add a Block called "Movie Widget" from "Common blocks"
5. Use the search field to search for a movie. Select it and save the post.
6. Voila! A movie widget of your favorite movie is displayed in the post.
